serviceMemory[0] - cycle counter
serviceMemory[1] - instruction pointer
serviceMemory[2] - cycle entry
serviceMemory[3] - end position of cycle

Supported commands:
input arg1 arg2
put arg2 value in arg1 cell of memmory

output arg
print agr to console

add arg1 arg2 arg3
take value from arg1 and add to value from arg2 and put in arg3 cell of memmory

switch arg1 arg2
switch arg1 and arg2 in memmory

equal arg1 arg2
return true if arg1 = arg2

repeat arg1 arg2
repeat arg2 count of next command agr times

inc arg
incriminates arg
